# Windows-style Calculator App

A simple Windows-like calculator built with HTML, CSS, and JavaScript.

## Features

- Standard calculator layout
- Basic operations: addition, subtraction, multiplication, division
- Clear, percent, decimal, and toggle sign support
- Keyboard input support

## Run locally

1. Open `index.html` in your browser.
2. Click buttons or use keyboard digits and operators.

## Files

- `index.html` — main UI structure
- `styles.css` — visual styling and Windows-inspired appearance
- `script.js` — calculator logic and interactions
